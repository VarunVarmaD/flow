# Flow

**Flow** is a minimalist, distraction-free To-Do list application designed to help you stay organized and focused.


## Features

-   **Minimalist Design**: Clean interface with a focus on your tasks.
-   **Dark Mode**: System-aware dark mode with a manual toggle.
-   **Persistence**: Tasks are saved automatically to your browser's LocalStorage.
-   **Responsive**: Works great on desktop and mobile.

## Getting Started

To run Flow locally, simply clone this repository and open `index.html` in your web browser.

```bash
git clone https://github.com/yourusername/flow.git
cd flow
# Open index.html in your browser
```

## Technologies

-   **HTML5**
-   **CSS3** (Variables, Flexbox)
-   **Vanilla JavaScript** (ES6+)
