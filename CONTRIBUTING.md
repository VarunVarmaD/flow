# Contributing to Flow

Thank you for your interest in contributing to Flow! We welcome contributions from everyone.

## Development Setup

1.  Clone the repository: `git clone https://github.com/yourusername/Flow.git`
2.  Open the project in your favorite editor (VS Code recommended).
3.  Open `index.html` in your browser to view the app.


1.  **Fork the repository**.
2.  **Create a new branch** for your feature or bug fix: `git checkout -b my-new-feature`.
3.  **Make your changes**. Keep the code simple and clean.
4.  **Commit your changes**: `git commit -m 'Add some feature'`.
5.  **Push to the branch**: `git push origin my-new-feature`.
6.  **Submit a pull request**.

## Code Style

-   Use 4 spaces for indentation.
-   Keep CSS variables in `style.css`.
-   Ensure the app works in both Light and Dark modes.

## Reporting Bugs

If you find a bug, please open an issue and describe:
-   What you expected to happen.
-   What actually happened.
-   Steps to reproduce the issue.
